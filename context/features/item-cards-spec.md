# Item Cards

## Overview

Item cards should have better spacing, border color status should be on the left border. Page sould support two view modes, one with grid cards and the other with list cards like the view toggle in the Google Drive.
Current sort by Name, Brand, Date and status sould resemble shadcn button group without horizontal padding, active button should be in cyan color.

## Requirements

- Organize card content in more logical UX friendly way
- Move coloured border from right to left
- Two view modes, one grid like and the other list like
- Cyan color on the "Assigned to" user and on the rent button
- Shadcn button group as search filters
- Cyan search filter active button
